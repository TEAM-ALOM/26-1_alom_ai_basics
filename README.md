# 26-1_alom_ai_basics

**아롬 AI 기초반**은 신입생 및 학부생을 대상으로 하는  
**파이썬 기반 머신러닝 입문 스터디**입니다.

NumPy·Pandas를 활용한 데이터 처리부터  
Scikit-learn 기반의 분류·회귀 모델,  
XGBoost·LightGBM과 같은 실전 모델,  
추천 시스템까지  
머신러닝에서 자주 사용되는 핵심 주제들을 **단계적으로 학습**합니다.

---

## 목표

- NumPy와 Pandas를 활용하여 머신러닝을 위한 데이터 전처리 및 분석 흐름을 이해합니다.
- Scikit-learn 기반 분류·회귀 모델을 직접 구현하고 평가 지표를 해석할 수 있도록 합니다.
- XGBoost, LightGBM 등 트리 기반 앙상블 모델의 동작 원리와 활용 방법을 익힙니다.
- 단순 구현을 넘어, 모델 선택 이유와 결과 해석까지 설명할 수 있는 수준을 목표로 합니다.
